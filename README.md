# SIAKAD-ITS Modern Theme

Chrome Extension untuk merubah tampilan SIAKAD-ITS (akademik.its.ac.id) menjadi lebih modern.

## Cara Install

1. Buka `chrome://extensions/`
2. Nyalakan **Developer mode** (pojok kanan atas)
3. Klik **Load unpacked**
4. Pilih folder `akademik-its-theme`
5. Selesai

## Cara Pakai

- Klik icon extension di samping address bar
- Geser toggle **Modern UI** untuk ON/OFF
- Status toggle tersimpan otomatis
